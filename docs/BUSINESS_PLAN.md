# 배드민국 사업계획서
### 대한민국 배드민턴 공인 MMR 플랫폼

---

## 1. 사업 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | **배드민국** (배드민턴 + 대한민국) |
| 슬로건 | 대한민국 배드민턴 공인 MMR |
| 형태 | PWA (모바일 웹앱) |
| 출시 | 2026년 7월 (live: badminguk.vercel.app) |
| 창업자 | 류OOOO 대표 |
| 현재 기술스택 | React 19 + Supabase + Vercel |

**핵심 아이디어**: 배드민턴에는 전국 단위 MMR(실력 점수)이 없다. 대회마다 주최자가 급수를 자율로 정하고, 상위 급수 선수가 하위 조에 출전하는 **샌드배깅**이 만연하다. 배드민국은 전화번호 기반 영구 신원 추적으로 이를 기술적으로 차단하는 **유일한 플랫폼**이다.

## 2. 시장 분석

### 배드민턴 인구
- 대한민국 배드민턴 등록 인구: **약 500만 명** (2024 대한배드민턴협회)
- 생활체육 배드민턴 동호회: 전국 약 3만 개
- 연간 생활체육 배드민턴 대회: **약 1,200~1,500개** (각 시군구 주최 포함)
- 평균 대회 참가비: 1인 3만~5만 원

### 경쟁사 분석

| 앱 | 강점 | 약점 | 배드민국 대비 |
|---|---|---|---|
| **스포넷** | 공인 급수 데이터 | 대회 운영 없음, UI 구식 | 파트너십 타깃 |
| **배프(배드파워)** | 대회 신청/운영 | MMR 없음, 샌드배깅 무방비 | 직접 경쟁 |
| **위꾹** | 종목 다양(테니스 등) | 배드민턴 특화 아님 | 차별화 명확 |
| **카카오스포츠** | 카카오 트래픽 | 기능 단순, 관심 낮음 | 포지션 격차 |

**결론**: MMR + 대회운영 + 반샌드배깅을 동시에 갖춘 플랫폼은 현재 존재하지 않는다.

## 5. 대회 유치 전략 (Go-to-Market)

### Step 1 — 개척대회 0→1 (2026년 7~8월)
**직접 주최 또는 지인 대회 디지털 전환**

> "무료로 대진표 + 스코어 + 성적 자동화 해드립니다"

- 인근 배드민턴 동호회 2~3개에 직접 연락
- 대회 운영을 배드민국으로 완전 대체 (종이 대진표 → 앱)
- **비용 없음** → 거절 이유 없음
- 대회 후기 + 영상 콘텐츠 SNS 배포 (바이럴 소재 확보)

### Step 2 — 지역 주최자 확보 (2026년 8~10월)
**주최자 커뮤니티 침투**

- 배드민턴 카페(다음/네이버) 주최자 섹션에 "AI 대회 운영 플랫폼 무료 제공" 게시
- 구장 관리자에게 영업: "손님 대회 → 배드민국 연동 = 구장 홍보 효과"
- 지역 배드민턴협회 대상 제안: "공식 앱 파트너 협약"

**목표**: 월 20개 대회 확보 (= 매출 기점)

### Step 3 — 배프/스포넷 파트너십 (2026 Q4~)
**경쟁이 아닌 연결**

- **스포넷**: 공인 급수 데이터 API 수신 → 배드민국 MMR과 연동
  - 제안: 배드민국이 생활체육 데이터를 역으로 스포넷에 공급 (Win-Win)
- **배프**: 대회 신청은 배프, 운영·스코어는 배드민국 → 기능 분리 협업

### Step 4 — 미디어 전략
- **유튜브 쇼츠**: 대회 명장면 자동 클립 + MMR 변화 그래프 (바이럴 콘텐츠)
- **배드민턴 인플루언서**: 상위 MMR 플레이어의 랭킹 공개 → 자연스러운 노출
- **전국 랭킹 발표**: 분기별 전국 MMR TOP 100 공개 → 언론 홍보

## 6. 반샌드배깅 = 핵심 해자

이것이 배드민국의 **가장 강력한 진입장벽**이다.

현재 배드민턴 커뮤니티 최대 불만:
> "C조인데 A조 실력으로 와서 다 쓸어가는 사람들 때문에 망했다"

배드민국의 기술적 해결책:
1. **전화번호 = 영구 신원** (탈퇴 후 재가입해도 복원)
2. **peak_grade는 오직 상승만 가능** (DB 트리거로 강제)
3. **하위급수 출전 신청 자체가 차단** (시스템 레벨)

배프·위꾹은 이 시스템이 없다. 따라서:
- "공정한 대회 = 배드민국"이라는 브랜드 포지셔닝이 가능
- 협회/주최자가 신뢰하면 배드민국이 사실상 표준이 됨
